Contact Center Dashboard

I am currently maintaining a dashboard I helped create for the university's contact center. This dashboard gives management the ability to see specific call volume/call types for all departments and contact center agents, for any specific date range. The dashboard also allows the user to save their call log filters. Due to the sensitivity of the contact center's data, below I only provide a demonstration of the framework used to build the dashboard (chartsJS), along with data similar to what the actual dashboard uses. While working on this project I have gained experienced and implemented the following skills and libraries: bootstrap, javascript,css,html,MVC framework, jquery,angularJS,C#,SQL, and especially the graph framework chartsJS. I built the front end javascript to work with the backend my manager created. My main task on this project was creating javascript functions with ajax calls to query our server, and correctly populate the graphs/tables with the returned data.

      Conference Room Scheduler

      This was my first real project. I designed a UI for reserving conference rooms in our building with the help of a co-worker. Before our project, printing the door signs for our conference room was a very tedious process of cutting and pasting an excel template. Now, with this project we simply launch a link, and the user gets a very neat, clean UI to work with. All they have to do is enter the name of the event, the contact infor, and the time range. I also added error proofing so that event overlapping wasn't possible. This project honed my skills in HTML,CSS,and JavaScript.
